Análise dos Resultados MO x CTF¶

Analisando-se o relatório de resumos de resultados para o modelo $CTF = a + b_1 MO + \varepsilon$, tem-se que:

  • $R^2 = 0,001674$: indica que não existe correlação linear ($R^2 \approx 0$) entre as variáveis, pois $0,1674\%$ da variação de $CTF$ são explicados pela variação de $MO$;
  • $P_{value} = 0,8494 (84,94\%)$ : indica que o relacionamento linear entre as variáveis $CTF$ e $MO$ não é significativo, pois não é menor que $5\%$.
  • O modelo proposto é $CTF = 77100,39 - 3,68 MO$
  • Conclui-se que o modelo proposto é inadequado para prever o comportamento da variável dependente $CTF$.

Análise dos Resultados (MO, EE) x CTF¶

Analisando-se o relatório de resumos de resultados para o modelo $CTF = a_3 + b_1 MO + b_2 EE +\varepsilon$, tem-se que:

  • A equação de regressão múltipla assume a forma: $CTF = 184,884 -28,84MO + 42.14EE$
  • Para a variável explicativa $MO$, tem-se que o $P_{value} = 0,295 > 0,05$. Este valor é maior que o valor para o teste de significância, que é $0,05$, indicando que não há relacionamento linear significativo entre as variáveis $CTF$ e $MO$;

  • Para a variável explicativa $EE$, tem-se que o valor $P = 0,202 > 0,05$, indicando que não há relacionamento linear significativo entre as variáveis $CTF$ e $EE$;

  • Portanto, pode-se concluir que nem a variável $MO$ e nem a variável $EE$ contribuem para o aumento da eficiência de previsão do modelo resultante da regressão linear múltipla. Por isso, deve-se excluir ambas as variável do modelo.
